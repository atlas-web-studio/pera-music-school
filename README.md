# Pera Music School Website

A full-stack website developed for Pera Music School.

## Tech Stack

- React (Vite)
- Node.js
- Express
- MongoDB
- Cloudinary
- Nodemailer

## Features

- Public website
- Admin dashboard
- Teacher management
- Program management
- Contact & Trial Session forms
- Email notifications
- Image uploads via Cloudinary

## Project Structure

```
backend/
frontend/
```

## Environment Variables

Create a `.env` file in the backend with the required environment variables.

## Development

Backend:

```bash
npm install
npm start
```

Frontend:

```bash
npm install
npm run dev
```
